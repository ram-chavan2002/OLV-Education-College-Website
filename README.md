 OLV Education College Website

This is a professional college website developed for a client project.  
The website represents OLV Education College and provides complete information about courses, faculty, admissions, and contact details.


# Features
- College Information Section
- Courses & Programs Details
- Faculty Details
- Admission Inquiry Form
- Contact & Location Details
- Responsive Design (Mobile + Desktop)


 #Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL


# Project Structure

- `/assets` → Images, CSS, JS
- `/admin` → Admin Panel 
- `/includes` → Common PHP files
- `index.php` → Main homepage
